【ソフトウェア名】Clock
【対応OS】Windows10 64bit
【製作者】大池陸斗




【本アプリケーションについて】
	本アプリケーションは時計を表示するアプリケーションです。時計の針は円弧のようなデザインです。各時計の針の一番色の濃い部分が現在時刻を示しています。
	また正確な時間を知りたい方のためにデジタル時計も表示できます。また、針の色を変更できる機能もついています。

【動作に必要なファイルおよびソフトウェア】
	本アプリケーションの動作にはCygwinの各種パッケージ,OpenGLのインストールが必要です。インストール方法は
	http://teacher.nagano-nct.ac.jp/ito/Springs_of_C/cyg64/index.htmlを参考にしてください。

【環境変数の設定】
	Cygwinのインストールフォルダ内のbinを環境変数名"Path"としてパスを通してください。
	Cygwinのインストールフォルダ内のusr\i686-pc-cygwin\sys-root\usr\binを環境変数名"Path"としてパスを通してください。

【インストール方法】	
	本ファイルが入っているディレクトリ内でコマンドプロンプトなどで"make"を実行します。
	なお警告が出る場合がありますが、プログラムの動作に問題はありません。j17404.exeができていれば成功です。
	

【実行方法】
	本ファイルが入っているディレクトリ内でコマンドプロンプトなどで"j17404.exe"を実行するか
	エクスプローラーなどからj17404.exeをダブルクリックで実行してください。
	
【操作方法】
	起動すると円弧の時計の針が表示されます。赤色が分針、ピンクが時針、黄色が秒針を示しています。

	・左クリックすることでデジタル時計とアナログ時計を切り替えることができます。
	・右クリックすることで時計の針の色をランダムに変更します。
	・ウィンドウの閉じるボタンを押すことでアプリケーションを終了します。
